## Quick Example

```typescript
import { Queue, Worker } from 'bunqueue/client';

// Create a queue
const queue = new Queue('emails');

// Add jobs
await queue.add('send-welcome', {
  to: 'user@example.com',
  template: 'welcome'
});

// Process jobs
const worker = new Worker('emails', async (job) => {
  await sendEmail(job.data.to, job.data.template);
  return { sent: true };
});

worker.on('completed', (job, result) => {
  console.log(`Email sent: ${job.id}`);
});
```

## Two Modes

**Embedded Mode** - Use as a library, no server needed:
```typescript
import { Queue, Worker } from 'bunqueue/client';
```

**Server Mode** - Run as a standalone server with HTTP/TCP API:
```bash
bunqueue start --data-path ./data/queue.db
```
