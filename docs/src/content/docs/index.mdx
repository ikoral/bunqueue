## Quick Start

<Tabs>
  <TabItem label="Embedded Mode">
    ```typescript
    import { Queue, Worker } from 'bunqueue/client';

    // Create a queue
    const queue = new Queue('emails', { embedded: true });

    // Add jobs
    await queue.add('welcome', {
      to: 'user@example.com',
      template: 'welcome'
    });

    // Process jobs
    const worker = new Worker('emails', async (job) => {
      await sendEmail(job.data);
      return { sent: true };
    }, { embedded: true });

    worker.on('completed', (job) => console.log(`✓ ${job.id}`));
    ```
  </TabItem>
  <TabItem label="Server Mode">
    ```bash
    # Start the server
    bunqueue start --data-path ./data/queue.db

    # Push a job
    bunqueue push emails '{"to":"user@test.com"}' --priority 10

    # Pull and process
    bunqueue pull emails
    bunqueue ack <job-id>
    ```
  </TabItem>
  <TabItem label="Docker">
    ```dockerfile
    FROM oven/bun:latest
    WORKDIR /app
    RUN bun add bunqueue
    COPY . .
    CMD ["bunqueue", "start"]
    ```

    ```bash
    docker run -p 6789:6789 -p 6790:6790 \
      -v ./data:/app/data \
      -e DATA_PATH=/app/data/queue.db \
      bunqueue
    ```
  </TabItem>
</Tabs>

## Features

<CardGrid>
  <Card title="Priority Queues" icon="bars">
    Process high-priority jobs first with configurable priority levels.
  </Card>
  <Card title="Delayed Jobs" icon="stopwatch">
    Schedule jobs to run at a specific time or after a delay.
  </Card>
  <Card title="Retries & Backoff" icon="approve-check-circle">
    Automatic retries with exponential backoff on failure.
  </Card>
  <Card title="Rate Limiting" icon="setting">
    Control job processing rate per queue or globally.
  </Card>
  <Card title="Stall Detection" icon="warning">
    Automatic recovery of jobs that become unresponsive.
  </Card>
  <Card title="Dead Letter Queue" icon="error">
    Advanced DLQ with metadata, filtering, and auto-retry.
  </Card>
  <Card title="Cron Jobs" icon="seti:clock">
    Schedule recurring jobs with cron expressions or intervals.
  </Card>
  <Card title="Parent-Child Flows" icon="document">
    Create job hierarchies with dependencies.
  </Card>
  <Card title="Progress Tracking" icon="up-caret">
    Real-time progress updates and job logs.

    HTTP callbacks for job events.
  </Card>
  <Card title="S3 Backup" icon="cloud">
